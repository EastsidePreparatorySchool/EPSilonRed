      • action

        public void action(IAction action)
        In addition to lines, a dashboard may also contain a list of actions. When the dashboard is to be updated, these actions are evaluated before the dashboard lines are composed. A typical use of such actions is to initialize some state variable, parts of which are subsequently displayed in dashboard lines and items. This can help avoid needless re-evaluation.
