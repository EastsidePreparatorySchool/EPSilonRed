Interface IBNO055IMU


  • public interface IBNO055IMU
    Interface API to the Adafruit 9-DOF Absolute Orientation IMU Fusion Breakout - BNO055 sensor.

    http://www.adafruit.com/products/2472, http://www.bosch-sensortec.com/en/homepage/products_3/9_axis_sensors_5/ecompass_2/bno055_3/bno055_4

      • getSystemError

        byte getSystemError()
        If getSystemStatus() is 'system error' (1), returns particulars regarding that error. See section 4.3.58 of the BNO055 specification.
        ResultMeaning
        0no error
        1peripheral initialization error
        2system initialization error
        3self test result failed
        4register map value out of range
        5register map address out of range
        6register map write error
        7BNO low power mode not available for selected operation mode
        8acceleromoeter power mode not available
        9fusion algorithm configuration error
        Asensor configuraton error

      • getSystemStatus

        byte getSystemStatus()
        Returns the current status of the system. See section 4.3.58 of the BNO055 specification.
        ResultMeaning
        0idle
        1system error
        2initializing peripherals
        3system initialization
        4executing self-test
        5sensor fusion algorithm running
        6system running without fusion algorithms

      • setPositionAndVelocity

        void setPositionAndVelocity(IBNO055IMU.Position position,
                          IBNO055IMU.Velocity velocity)
        Atomically sets the current position and / or velocity of the sensor. It is explicitly OK to call this method while the acceleration integration loop is executing.
        Parameters:
        position - If non-null, the current sensor position is set to this value. If null, the current sensor position is unchanged.
        velocity - If non-null, the current sensor velocity is set to this value. If null, the current sensor velocity is unchanged.
        See Also:
        startAccelerationIntegration(Position, Velocity), getPosition(), getVelocity()

      • startAccelerationIntegration

        void startAccelerationIntegration(IBNO055IMU.Position initalPosition,
                                IBNO055IMU.Velocity initialVelocity)
        Start (or re-start) a thread that continuously at intervals polls the current linear acceleration of the sensor and integrates it to provide velocity and position information. A default polling interval of 100ms is used.
        Parameters:
        initalPosition - as in setPositionAndVelocity(Position, Velocity)
        initialVelocity - as in setPositionAndVelocity(Position, Velocity)
        See Also:
        startAccelerationIntegration(Position, Velocity), setPositionAndVelocity(Position, Velocity), getLinearAcceleration()
