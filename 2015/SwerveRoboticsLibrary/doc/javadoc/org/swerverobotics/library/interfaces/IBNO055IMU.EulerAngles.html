    IBNO055IMU

    public static class IBNO055IMU.EulerAngles
extends Object
    Instances of EulerAngles represent a direction in three-dimensional space by way of rotations. Units are as specified in sensor initiation. Angles are in rotation order (heading, then roll, then pitch) and are right-handed about their respective axes.
