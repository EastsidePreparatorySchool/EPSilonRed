org.swerverobotics.library

Class ClassFactory

  • Object
    • ClassFactory

  • public final class ClassFactory
extends Object
    ClassFactory provides static methods for instantiating objects within the Swerve Robotics Library. The point here is to separate clients from having to know the actual class names or package structures in which various object implementations are stored; these are often internal, and thus should not be exposed to clients, as they are not stable from release to release.
